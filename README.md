# Hangman Game 🎯

This is a console-based Hangman game written in Python.  
It's part of the **100 Days of Code** course by Angela Yu.

📅 **Progress: Day 7**

## 💡 How to Run
```bash
python main.py
```

🗂️ Project Structure
- main.py: game logic
- hangman_words.py: word list
- hangman_art.py: ASCII logo and hangman stages
